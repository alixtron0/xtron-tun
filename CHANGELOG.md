# Changelog

تمام تغییرات مهم این پروژه در این فایل مستند می‌شود.

فرمت بر اساس [Keep a Changelog](https://keepachangelog.com/fa/1.0.0/) است.

## [1.0.0] - 2026-02-11

### افزوده شده
- اسکریپت نصب یک‌خطی
- ماژول سرور خارج (Kharej) با قابلیت‌های:
  - راه‌اندازی SOCKS5 Proxy با GOST v3
  - مدیریت تونل‌ها
  - صادرات تنظیمات به ZIP
  - Ping Test و مانیتورینگ
  - حذف و پاکسازی تونل‌ها
- ماژول سرور ایران (Iran) با قابلیت‌های:
  - وارد کردن تنظیمات از ZIP یا دستی
  - Port Forwarding با GOST و socat
  - مدیریت چندین Port Forward همزمان
  - نمایش وضعیت و لاگ‌ها
  - حذف کامل تنظیمات
- کتابخانه توابع کمکی (utils.sh)
- رابط کاربری فارسی و زیبا
- سیستم لاگ‌گیری پیشرفته
- مستندات کامل فارسی و انگلیسی
- پشتیبانی از Ubuntu, Debian, CentOS, RHEL
- راه‌اندازی خودکار سرویس‌های systemd
- پیکربندی فایروال خودکار

### تغییر یافته
- بهبود عملکرد Spinner
- بهینه‌سازی نصب بسته‌ها

### برطرف شده
- رفع باگ‌های جزئی در Validation
